# Official rule text source

## Intended source of truth

**U.S. Coast Guard, *Navigation Rules and Regulations Handbook*** (COMDTINST M16672.2D, as amended).

This U.S. government work is **public domain** in the United States. The `officialText` fields in this project are maintained to align with that handbook (Inland/International distinctions where applicable follow the same publication).

## What we do not use as a copy source

We do **not** copy rule text from IMO commercial publications (e.g. IMO-published rule books sold as copyrighted editions). Doing so would create commercial copyright exposure unrelated to the underlying treaty text.

## Audit

Periodically compare stored `officialText` strings against the current USCG handbook PDF (or successor instruction) and update if the Coast Guard revises wording.

## Site copy

User-facing attribution (footer and rule pages) states that rule text is sourced from the USCG handbook as a public domain U.S. government publication.

## Illustrations

Diagrams, light sectors, and other graphics in this app must be **original** or licensed; do not trace diagrams from IMO or USCG copyrighted layout/illustration pages where those works are not PD.
